═══════════════════════════════════════════════════════════════════════════════
                           GETTING STARTED
═══════════════════════════════════════════════════════════════════════════════

1. QUICK START (5 minutes):
   $ cd leadgen
   $ pip install -r requirements.txt
   $ python cli.py search-keywords --keywords "Sales" --limit 10
   
   See QUICKSTART.md for detailed guide

2. CLI TESTING (Local):
   $ python cli.py search-industry --industry "Tech" --location "SF"
   $ python cli.py search-company --company "Google"
   $ python cli.py batch --file queries.txt --output results.json

3. WEB APP (React):
   $ cd reactjs
   $ npm install
   $ npm start
   Opens http://localhost:3000

4. ANDROID APP:
   Follow complete guide in ANDROID_SETUP.md
   - Create Android project with Chaquopy
   - Copy leadgen_core/ to Python path
   - Setup WebView with React UI
   - Build and run on device
