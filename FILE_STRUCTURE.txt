  Classes:
    • Lead              - Lead data structure
    • LeadStatus        - Enum: new, contacted, qualified, etc.
    • ScrapeSource      - Enum: linkedin, gmap, scrapegraph, manual
  
  Features:
    • Automatic lead scoring (0-100)
    • Data validation
    • JSON/CSV export
    • Immutable status tracking

  Class: LeadScraper
  Methods:
    • search_by_industry()    - Find leads by industry
    • search_by_company()     - Get company employees
    • search_by_location()    - Search by geography
    • search_by_keywords()    - Advanced search
    • batch_search()          - Multiple queries
  
  Features:
    • Multi-source aggregation
    • Automatic deduplication
    • Progressive enrichment
    • Fallback strategy

linkedin_scraper.py (5 KB) - LINKEDIN INTEGRATION
  Class: LinkedInScraper
  Methods:
    • search_company()           - Company info
    • search_person()            - Find contacts
    • extract_leads_from_company()
    • extract_leads_from_search()
  
  Data Captured:
    • Company info, size, industry
    • Contact names, titles
    • LinkedIn profile URLs

gmap_scraper.py (3 KB) - GOOGLE MAPS INTEGRATION
  Class: GMapScraper
  Methods:
    • search_business()              - Business search
    • extract_leads_from_results()   - Parse results
  
  Data Captured:
    • Business name, address
    • Phone, website, category
    • Ratings, reviews
    • Estimated company size

scrapegraph_adapter.py (4 KB) - SCRAPEGRAPH AI INTEGRATION
  Class: ScrapeGraphAdapter
  Methods:
    • scrape_website()     - Extract structured data
    • enrich_lead()        - Enrich individual lead
    • bulk_enrich_leads()  - Batch enrichment
  
  Used For:
    • Missing emails
    • Missing phone numbers
    • Industry classification
    • Revenue estimation

enricher.py (6 KB) - MULTI-SOURCE ENRICHMENT
  Class: LeadEnricher
  Methods:
    • enrich_lead()        - Single lead from multiple sources
    • enrich_leads()       - Batch processing
    • _merge_leads()       - Non-destructive merging
  
  Strategy:
    1. Try LinkedIn (primary)
    2. Fill gaps with Google Maps
    3. Use ScrapeGraphAI (fallback)
    4. Preserve existing quality data

                         HOW TO USE EACH FILE
═════════════════════════════════════════════════════════════════════════════

FOR DESKTOP TESTING:
  1. Install: pip install -r requirements.txt
  2. Run: python cli.py search-keywords --keywords "tech"
  3. Export: --output results.json

FOR WEB DEVELOPMENT:
  1. Install: npm install (in reactjs/)
  2. Start: npm start
  3. Code: Edit files in reactjs/src/
  4. Build: npm run build

FOR MOBILE DEVELOPMENT:
  1. Read: ANDROID_SETUP.md
  2. Create: Android Studio project
  3. Copy: leadgen_core/ folder
  4. Code: android/src/main/java/
  5. Build: gradle assembleRelease

FOR CUSTOM IMPLEMENTATION:
  1. Study: ARCHITECTURE.md
  2. Extend: Add new scraper classes
  3. Modify: Adjust scoring logic
  4. Test: Run with CLI tool
  5. Deploy: Follow DEPLOYMENT.md
